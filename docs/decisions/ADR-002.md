# ADR-002: Simulation First

Date: 2026-06-04
Status: Accepted

## Context

RAZON handles financial decisions where unsafe execution, overconfidence, and
unverified performance can cause real loss.

## Decision

Simulation and demo behavior are the default. Live execution and automatic
trading remain disabled until data, analysis, risk, journal, backtesting, and
demo validation are complete.

## Consequences

Development can proceed safely. Any future live execution path must require
explicit configuration, risk approval, audit logging, and emergency stop
support.
